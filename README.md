# MemoLeak-Chan
指定したサイズ分だけメモリを確保します

## 対応OS
Windows 64bit

おそらく他のOSでもビルドすれば実行できると思います

## How to Use
### バイナリをダウンロード
1. Releaseからビルドファイルをダウンロードします
2. MemoLeak-Chan.exeを実行します
3. 確保したいサイズを指定します

### ソースコードからビルド
1. リポジトリをクローンします
2. ソースコードを次のようにビルドします
   `cl MemoLeak-Chan.c /Od`
3. 生成された実行ファイルを実行します
4. 確保したいサイズを指定します
